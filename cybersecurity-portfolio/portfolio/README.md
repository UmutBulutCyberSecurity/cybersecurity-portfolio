# Cybersecurity Portfolio — Ahmet Umut Bulut

A collection of hands-on security labs focused on **detection, log analysis, and incident investigation**. Each project documents not just *what* I did, but *why* — how an attack works and, more importantly, how a defender detects and stops it.

> **Ethics note:** Every lab in this portfolio is carried out in isolated, virtualised environments against intentionally vulnerable or self-owned targets, for the purpose of learning and defensive research. Nothing here is a step-by-step guide for attacking real systems.

## Projects

### 1. [Host-Based Intrusion Detection with OSSEC](projects/01-ossec-hids.md)
Deployed OSSEC HIDS across a multi-VM lab, ran controlled attacks, and verified that intrusions, new accounts, RDP activity, and file changes were detected and logged.
`Detection engineering` `HIDS` `Log analysis` `File integrity monitoring`

### 2. [Attack Detection & Log Analysis (Metasploitable 2)](projects/02-attack-detection.md)
Gained access to a vulnerable host via default credentials, then switched to the defender's side to investigate the intrusion through authentication logs and evidence ranking.
`Incident investigation` `auth.log` `Brute-force detection`

### 3. [Penetration Test Report — Healthcare Network (Lab)](projects/03-pentest-report.md)
Produced a full client-style penetration test report: scope, findings, business impact, and prioritised remediation for an EternalBlue (MS17-010) compromise.
`Reporting` `Remediation` `Risk communication`

### 4. [OSINT & Passive Reconnaissance](projects/04-osint-recon.md)
Used Shodan and OSINT techniques to map an organisation's internet-facing footprint from public data and identify infrastructure concentration risk.
`OSINT` `Attack-surface mapping` `Threat intelligence`

### 5. [Social Engineering / Phishing Simulation](projects/05-social-engineering.md)
Ran an authorised phishing exercise to study how trust and urgency drive successful attacks, and documented the defences that actually reduce that risk.
`Human-layer security` `Phishing` `Security awareness`
